ros2 run turtlesim turtlesim_node - start a turtle node

ros2 run turtlesim turtle_teleop_key - start a node to send commands to move the turtle in turtle sim node

ros2 pkg create my_package --build-type ament_python --dependencies rclpy - create new ros package with python build type and rcply dependency

colcon build - build all the packages in the ros workspace

rqt_graph - visualize node interactions

ros2 topic list - list all active topics

ros2 topic echo <topic_name> - subscribe to topic

ros2 topic info <topic_name> - show topic info

ros2 interface show <interface> - display information of the interface
